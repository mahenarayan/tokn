# Example Gallery

This gallery shows realistic Tokn workflows using fixtures already checked into the repository.

Run the commands from the repository root after:

```bash
npm install --cache .npm-cache
npm run build
```

The examples use `node dist/cli.js` so they work without `npm link` or npm publication.

## How To Use The Gallery

Each example is intentionally small:

- one problem statement
- one or two commands
- what the output is meant to answer
- how that workflow maps to real engineering usage

Treat these as executable starting points, not static screenshots.
